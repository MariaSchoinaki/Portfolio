roomie

Roomie is a distributed Airbnb booking system built with Java and Kotlin, leveraging the MapReduce framework for efficient data processing and storage. It features a backend for managing Airbnb listings, reservations, and analytics, and a frontend Android app for customers to search, book, and review accommodations. Roomie supports multi-worker scalability, enabling fast data processing while providing a seamless experience for both managers and customers.
